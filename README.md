# Syntara

AI Prompt Engineering Toolkit — generate, score, build, and compare prompts.

## Features

- **Prompt Generator** — generate and improve prompts with multiple modes
- **Prompt Scoring Engine** — score prompts on specificity, clarity, structure, constraints, output format, and role
- **Modular Prompt Builder** — assemble prompts with toggleable blocks
- **Prompt Library** — search, edit, and version-track saved prompts
- **A/B Compare** — side-by-side prompt scoring
- **LocalStorage persistence** — saves prompts locally (backend-ready)

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- Vitest

## Getting Started

```sh
# Clone the repo
git clone <YOUR_GIT_URL>
cd Syntara

# Install dependencies
npm install

# Start dev server
npm run dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
| `npm run test` | Run tests |
